# 🚀 Pregunta AskAI - Fullstack Dockerized App

A fullstack AI-style chat application built with **React (Vite)** frontend and **Python backend**, fully containerized using **Docker & Docker Compose**.

---

## 📦 Tech Stack

### Frontend
- React (Vite)
- JavaScript (ES6+)
- CSS Modules / Components

### Backend
- Python (FastAPI / Flask style API)
- REST API architecture

### DevOps
- Docker
- Docker Compose
